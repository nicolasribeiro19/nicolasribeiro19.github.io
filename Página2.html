Média Aritmética

Definição de Média Aritmética

Média aritmética é a soma de vários valores e dividido pelo total deles. Ou seja, o resultado dessa divisão equivale a um valor médio entre todos os valores.

Média Ponderada

Definição de Média Ponderada

A média ponderada leva o peso de cada informação em consideração no cálculo, seja ele um valor atribuído, seja uma quantidade.
Na média simples, os valores são somados e divididos pela quantidade de termos adicionados. A média ponderada é calculada por meio do somatório das multiplicações entre valores e pesos divididos pelo somatório dos pesos.
Vamos, por meio de exemplos, demonstrar os cálculos envolvendo a média ponderada.


Fórmula


Onde,
Mp: Média aritmética ponderada
p1, p2,..., pn: pesos
x1, x2,...,xn: valores dos dados

Média Geométrica

Definição da Média Geométrica

A média geométrica é definida, para números positivos, como a raiz n-ésima do produto de n elementos de um conjunto de dados.
Assim como a média aritmética, a média geométrica também é uma medida de tendência central. É usada com mais frequência em dados que apresentam valores que aumentam de forma sucessiva.

Fórmula


Onde,
MG: média geométrica
n: número de elementos do conjunto de dados
x1, x2, x3, ..., xn: valores dos dados

Exemplo
